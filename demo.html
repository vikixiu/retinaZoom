Overview

Retina zoom is jQuery plugin which offers an Apple like zoom style.
It comes from Apple-like Retina Effect With jQuery.. But now is much more easily to use, and works well both in IE and morden broswers.

How to use

HTML Parts:

1. Add below right after the <head> tag, let IE8 Display in IE7 Standards mode.

<meta content="IE=EmulateIE7, IE=9" http-equiv="X-UA-Compatible" />

2. Put the origin image before '#retina'. Then wrap them with a holder DIV. Take note of the 'longdesc' attribute, it offers zoom image's information so must be defined as "zoomImageUrl|zoomImageWidth|zoomImageheight"

<div class="holder">
	<img src="img/psbe.jpg" width="335" height="188" alt="" longdesc="img/psbe.jpg|1200|674" />
	<div class="retina" ></div>
</div>

3. Include JS files

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js">
</script>
<script type="text/javascript" src="jquery_mousewheel_plugin.js"></script>
<script type="text/javascript" src="DD_roundies_0.0.2a.js"></script>
<script type="text/javascript" src="jquery.retina.js"></script>
CSS Parts
.holder{
	position:relative;
}

.retina{
	/* The Retina effect */
	border:2px solid white;

	/* Positioned absolutely, so we can move it around */
	position:absolute;
	height:120px;
	width:120px;

	/* Hidden by default */
	display:none;

	/* A blank cursor, notice the default fallback */
	cursor:crosshair;
	
	/* CSS3 Box Shadow */
	-moz-box-shadow:0 0 5px #777, 0 0 10px #aaa inset;
	-webkit-box-shadow:0 0 5px #777;
	box-shadow:0 0 5px #777, 0 0 10px #aaa inset;
	
	/* CSS3 rounded corners */
	-moz-border-radius:200px;
	-webkit-border-radius:200px;
	border-radius:200px;
	
	background-repeat:no-repeat;
	}
Demo

default

$('#retinaDefault').retina();

Stop retina zoom

$('.retina').retina({wheel:false});

Limit retina size

$('.retina').retina({sizelimit:{max:200,min:80}})
